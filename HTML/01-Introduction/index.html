<!DOCTYPE html> defines whether the document is html5 document or not.
  • It determines the document type and help browsers to display Webpages properly.
  • It should be defined at the top of the document before any HTML tags.
  • It is not case sensitive.
<html> is the Root element of the document
<head> element contains the meta information about the document.
<title> element defines the tile the document that or visible on the browser tab.
<body> contains the actual content that we see in the webpage. it the the container of all elements of the document body.
h1,h2,h3,h4,h5,h6 are the available Heading Tabs
<openTag> Content </closeTag> Any HTML element starts with the opening tag and ends with the closing tag that may or may not contain contents.
